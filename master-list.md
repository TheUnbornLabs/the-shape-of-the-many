# Master List — The Shape of the Many

The permanent numbered index of all 100 curriculum items. **Numbering is fixed and never changes.** Each item maps to exactly one book and never moves between books. Each row carries a slug (its folder name) and a one-line hook that distinguishes it from its neighbours (guards against near-duplicate chapters).

Retirement rule: if an item is ever merged, split, or dropped, mark the slot here (e.g. `037 retired → merged into 041`) and keep the number reserved. Never renumber. New items append at the end.

Folder convention: `book-N-.../NNN-slug/` containing `chapter.md` + `references.md`.
